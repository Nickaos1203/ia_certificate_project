# ia_certificate_project

# Contexte :
Pour l'obtention de la certification "Développeur IA" de Simplon (titre RNCP de niveau 6), plusieurs blocs de compétences doivent être validés :
- Bloc 1 : Réaliser la collecte, le stockage et la mise à disposition des données d'un projet en intelligence artificielle ;
- Bloc 2 : Intégrer des modèles et des services d'intelligence artificielles ;
- Bloc 3 : Réaliser une application intégrant un service d'intelligence artificielle.
